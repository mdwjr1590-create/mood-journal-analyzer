# Mood Journal Analyzer

This project will be a Python program that analyzes short journal entries entered by the user. The program will let the user type in daily reflections, then look for patterns in mood-related words and give a simple summary of emotional trends over time. The goal is to combine file handling, functions, and data analysis in a project that is more advanced than the earlier labs but still manageable. The program may also show the most common words used in entries and organize entries by date.

## Planned Functions

### 1. add_entry()
This function will allow the user to type a journal entry and save it to a file along with the date. It will make it possible to build a collection of entries over time.

### 2. analyze_mood()
This function will examine saved journal entries and count positive, negative, or neutral mood-related words. It will then give the user a simple summary of the emotional tone of their writing.

### 3. word_frequency_report()
This function will calculate how often certain words appear in the journal entries. It will help the user see repeated themes or common words in their reflections.
